# Task Tracker

Angular to do list app with json-server

# Run Application 
```
$ git clone https://github.com/JaninduChanuka/Task-Tracker.git
$ cd Task-Tracker
$ npm install

//run the server
$ npm run server (It will run on 'http://localhost:5000')

//run the app
$ ng serve

Navigate to 'http://localhost:4200/'

```
